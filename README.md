# Ninja Man: Quest for the Shuriken, Iteration 1
First iteration of CS315 Final Project by Landon Hammond and Nate Slagter

### How to Play: 
* Move right: "right" or "d"  
* Move left: "left" or "a"  
* Jump: "spacebar"
* Attack: "up" or "w"
* Dodge: "down" or "s" (1.5 second cooldown)
